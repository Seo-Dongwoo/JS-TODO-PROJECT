# Javascript를 이용한 TODO-LIST Project.


### * 주요기능 *
1. input창에 todo를 추가한다.
2. 체크 버튼을 누르면 todo가 흐려지면서 체크 효과
3. 쓰레기통 버튼을 누르면 todo를 삭제
4. All, Completed, UnCompleted로 todo의 상태를 분류 가능
5. LoaclStorage를 이용한 key,value 저장 및 삭제
